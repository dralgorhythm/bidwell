# UI Design Documentation

## Design Principles
- **Premium & Clean**: Minimalist, content-focused.
- **Accessible**: High contrast, keyboard navigable.
- **Dark Mode First**: "Hacker/Engineer" aesthetic.

## Design System

### Typography
- **Sans**: `Geist Sans` (Body, Headings)
- **Mono**: `Geist Mono` (Code, Data)

### Color Palette
- **Background**: White / Black
- **Surface**: Neutral-50 / Neutral-900
- **Accent**: Blue-600 / Blue-400

### Components
- **Radius**: `rounded-lg`
- **Spacing**: Multiples of 4px.

## Page Layouts
- **Landing**: Grid layout for experiments/services.
- **Detail**: Breadcrumb navigation + Content area.
